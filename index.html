NGINX Helm Chart

A Helm chart for deploying NGINX on Kubernetes.

Installation

helm install my-nginx ./nginx-chart

Configuration

The following table lists the configurable parameters:

  • replicaCount: Number of replicas (default: 1)
  • image.repository: Image repository (default: nginx)
  • image.tag: Image tag (default: stable)
  • service.type: Service type (default: ClusterIP)
  • service.port: Service port (default: 80)
